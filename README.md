# Trabajo Integrador Final de Frontend: Whatlorant

Aplicación web, tipo WhatsApp, desarrollada en React. Incluye login personalizado, lista de chats, modo claro/oscuro, vista de conversaciones con respuestas en tiempo real, perfil de usuario y diseño responsive, adaptable a distintos dispositivos.

El trabajo esta hecha en base a una temática inspirada en Valorant. Donde la persona logueada participa como el jefe/a dentro de la dinámica de la aplicacion web. Para este resultado se tomo como inspiracion un poco del lore del juego y la selección de agentes.

> Desafío elegido

Se eligió construir una aplicacion web tipo WhatsApp para aplicar lo aprendido:

* Componentización en React
* Manejo de estado global y local
* Navegación mediante rutas
* Implementación de formularios
* Diseño responsive y accesibilidad
